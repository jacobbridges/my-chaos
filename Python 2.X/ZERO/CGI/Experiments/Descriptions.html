Python Script Viewer
SOURCE

The problem with having a web site server execute CGI is that you can never look at the program itself. The particular program is designed to get around that problem. The script will display a HTML form if there was no query string and possibly an error message if the file was invalid. Otherwise, the Python file is printed to your web browser as text.

IP Address
SOURCE

As you will notice, most of these files are just experiments. One of the best ways to learn is to experiment, and that is the idea behind this "playground." The webmaster of this site is trying to gain skills that will make programming CGI applications much easier in the future. The file to the left dynamically shows you your IP address -- something that should be unique to your computer. This would allow the server (the computer that your computer connected to) know who you are to a certain extext and provide you with dynamic content based on what is learned about you.

Directory Contents
SOURCE

This concept behind this program can be used in several different applications. First, being able to display what is in a directory shows that a script could be written to show the directory in any format a programmer might want. However, the script could also be modified to show the contents of other directories as well -- possibly even at the user's request. There are probably even more interesting ideas that people could and have put to work using the power of dynamic directory evaluation.

CGI Test
SOURCE

This program allows anyone to test what cgi_module.py and cgi_module.pyc accomplish for a CGI program. cgi_test.py is very small and should not be too difficult for someone to understand. To play around with this script, just add a valid or invalid query string to the end of the URL (example: ?country=Mars&state=Confused).

CGI Module
SOURCE

CGI Module is not designed to be executed directly. It is meant as a helper for other CGI programs. Is can export a query string and a query string in the form of a dictionary. It also has some functions that are helpful when trying to show output in a user's browser. Python already has CGI resources in its toolbox; CGI Module is just a lighter version for the same purpose.

Aens Time
SOURCE

Aens is a fictional universe with a fictional measure of time. Betas are like seconds and milliseconds combined. Minutes and hours are represented with alphas. There are 1,000 betas in an alpha and 1,000 alphas in a day. There a 7 days in a week and 4 weeks in a month. There are 3 months in a season and 4 seasons in a year. What does it all mean? There are 336 days in a year. No, they do not live longer; they just have more birthdays.
